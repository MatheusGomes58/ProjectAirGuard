# 📡 Monitoramento da Qualidade do Ar

Este projeto Open Source foi desenvolvido para **monitorar e gerenciar a qualidade do ar** em um determinado ambiente, garantindo **integração, segurança e conforto**. Utiliza sensores de qualidade do ar conectados a um sistema baseado em **IoT e Machine Learning** para fornecer insights em tempo real.

## 🚀 Funcionalidades

- 📊 **Monitoramento em Tempo Real**: Coleta e exibição de dados como temperatura, umidade, CO₂ e outros poluentes.
- 🔔 **Notificações e Alertas**: Envio de alertas caso a qualidade do ar atinja níveis críticos.
- 📡 **Integração IoT**: Suporte para sensores de qualidade do ar conectados via ESP32.
- 📈 **Análise Inteligente**: Utiliza Machine Learning para classificar o clima e prever variações na qualidade do ar.
- ☁️ **Armazenamento na Nuvem**: Registros históricos armazenados no Firebase Firestore.
- 🌍 **Dashboard Web**: Interface intuitiva para visualização e gerenciamento dos dados.

## 🛠️ Tecnologias Utilizadas

- **Backend**: FastAPI, Firebase Firestore
- **Frontend**: React.js
- **IoT**: ESP32, Sensor DHT11
- **Machine Learning**: Google Notebooks, Scikit-Learn
